# Security Awareness Evaluation Web Page

The **Security Awareness Evaluation Web Page** is a dynamic and user-friendly platform developed using React.js, Material UI, Node.js, and Express.js. This web application serves as a tool for assessing security awareness based on the 'ISO/IEC 27001 - 27002:2013' standard. It enables users to fill out a security awareness evaluation form, view graphical results, and receive a customized report via email. This project showcases my expertise in web development and highlights my capabilities for creating impactful applications.

## Features

- **Security Awareness Evaluation Form:** The web page presents a user-friendly form for evaluating security awareness. Users can answer a set of questions aligned with the 'ISO/IEC 27001 - 27002:2013' standard. The form is designed using the Material UI framework, ensuring an intuitive and visually appealing user experience.

- **Graphical Result Display:** After completing the evaluation, users can view their security awareness results in a graphical format. This visual representation provides users with a clear understanding of their strengths and areas that may require improvement.

- **Custom Report via Email:** Users have the option to receive a personalized security awareness report via email. The web page incorporates a feature that allows users to enter their email addresses. Upon submission, a custom report is generated based on the evaluation results and sent to the user's provided email address.

---

## Technology Stack

- **React.js:** The front-end of the web page is built using React.js, a powerful JavaScript library for building interactive user interfaces. React.js enables the creation of dynamic components and a smooth user experience.

- **Material UI:** The Material UI framework is employed to design a sleek and responsive user interface. It provides a consistent and modern design language, enhancing the overall look and feel of the web page.

- **Node.js:** The back-end of the application is powered by Node.js, allowing for efficient handling of server-side operations. Node.js enables non-blocking I/O, making the application responsive and scalable.

- **Express.js:** The web application utilizes Express.js, a minimal and flexible Node.js framework, to handle routing, API requests, and email sending functionalities. Express.js streamlines the development of the server-side components.
